
🎄 Jogo da Memória - Especial de Natal 🎅

Um jogo da memória simples, divertido e com temática natalina desenvolvido em HTML, CSS e JavaScript. O objetivo é encontrar todos os pares de cartas idênticas.

O jogo possui uma fase única e direta: assim que o jogador consegue encontrar todos os pares, a mensagem de vitória é exibida na tela.
🚀 Funcionalidades

    Tema Natalino: Cartas com elementos visuais de Natal (Papai Noel, Árvores, Presentes, etc.).

    Mecânica Clássica: Clique para virar as cartas e encontrar os pares.

    Fase Única: Fluxo rápido e objetivo.

    Tela de Vitória: Mensagem dinâmica de "Você Venceu!" assim que a fase é concluída.

🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando o trio básico do desenvolvimento web frontend, sem a necessidade de frameworks externos:

    HTML5: Estruturação das cartas e elementos da página.

    CSS3: Estilização, layout em Grid/Flexbox e animações de virada de carta.

    JavaScript (ES6): Lógica do jogo (shuffling/embaralhamento, checagem de pares e validação da vitória).

🎮 Como Jogar

    Abra link em qualquer navegador web moderno.

    Clique nas cartas para virá-las.

    Se as cartas forem iguais, elas permanecem viradas.

    Se forem diferentes, elas se viram de volta após um breve período.

    Encontre todos os pares para concluir a fase e ver a mensagem de Venceu! 🎉
